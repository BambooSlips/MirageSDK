# Mirage SDK

**Mirage** is a cross-platform, cross-language 3D engine SDK for games and interactive applications. It currently provides both **C++** and **Python** interfaces, attempting to deliver a consistent and efficient development experience.

It now stably supports **Windows** and **Linux**, with core features covering 3D rendering, skeletal animation, spatial audio, UI systems, and input management. But it is still under active development. Certain major features—such as lighting systems and physics engines—are not yet implemented and are planned for future releases. For the time being, we distribute it as **pre‑compiled binary libraries**—no complicated dependency handling required, just download and use.
